알지보드 4.x README (http://rgboard.com)
-------------------------------------------------------------------------------

1. 소개

프로그램명 : 알지보드
제작자 : 윤범석(master@rgboard.com)
홈페이지 : http://rgboard.com (알지보드홈페이지)

알지보드는 윤범석(master@rgboard.com)이 2003년 6월 개발한 무료공개 게시판으로 
기존 무료게시판이 무료라는 잇점이 있는 반면 저작권등 자유스럽지 못한 부분에 
보다 자유가 보장되는 게시판을 개발하려는 취지로 개발을 하게되었다.
일부소스는 적수보드등 기존 GPL프로그램의 라이브러리 또는 일부 루틴을 참고하여 
사용하거나 개작하여 사용하였으며 그외 본인이 직접 작성한 프로그램이다.

2. 동작환경

일반적인 리눅스 환경, RedHat등 APACHE,PHP,MySQL 또는 Cubrid 가 지원되는 환경이다.
아직 특수한 환경에서테스트 되어진적이 없으며 테스트가 필요하다.
Apache 와 PHP, MySQL 또는 Cubrid 가 설치되어 있는 서버라면 약간의 손질만 보면 운영이
가능 하리라 생각이 되어진다.
단 이 부분은 아직까지는 사용자들의 노력에 달려 있다고 생각된다.

3. 주의사항

윈도우 환경에서 테스트되어진 적이 없기때문에 사용을 원하실 경우 직접 테스트해본후
테스트 결과를 master@rgboard.com 으로 통보하여 시기 바랍니다.

4. 배포

알지보드의 공식적인 배포처는 http://rgboard.com 의 자료실에서 배포를 하되
필요한경우 타사이트에서 배포할수 있다.
정보의 공유를 위하여 배포시 되도록이면 출처를 밝혀주면 고맙겠다.

5. 설치방법

알지보드는 FTP또는 TELNET상 약간의 설정과 웹 인터페이스의 installer 를 제공한다.

받은 파일의 압축을 풀어 계정에 올린후 
rg4_data 디렉토리를 권한설정을 707 또는 777로 변경후 

브라우저에서
http://your-domain/알지보드설치디렉토리/rg4_admin/install.php
으로 데이타베이스정보를 입력, 관리자 정보를 입력후 로그인 화면이 
나오면 로그인하면 관리자 모드로 들어갈수 있다.

6. 개발자의 의무

개발자는 설치 및 운영에 있어 필요한 부분을 제공 하기는 하나 이는 의무에 속하지는
않는다.
또한 알지보드를 사용하는중 유실된 데이타에 대해서는 책임을 지지 않는다.
